# Тестовое задание: Компонент новостного блока для CDEK

Небольшое React-приложение с универсальным компонентом новостного блока. Компонент получает данные из тестового API и отображает новости с пагинацией и разными состояниями (загрузка, ошибка, пусто).

## Запуск проекта

Требования: Node.js и npm.

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите сервер для разработки:**
   ```bash
   npm run dev
   ```

3. **Откройте браузер:**
   Перейдите по адресу, указанному в консоли (обычно `http://localhost:5173`).

## Что внутри

- На странице показаны три варианта компонента: новости компании, бизнес-рубрика и пустое состояние.
- Пагинация реализована кнопками «назад/вперёд».
- Стили и адаптивность сделаны по макету из Figma.

## Технологический стек

- React + Vite + TypeScript
- CSS Modules
- Framer Motion
- Tabler Icons
- date-fns

## Реализовано (помимо базового)

- Кэширование страниц в `localStorage` (с фоновым обновлением данных).
- Отмена запросов при смене страницы (AbortController).
- Ленивая загрузка изображений (`loading="lazy"`).
- Skeleton во время загрузки.
- Типизация ответов API (`src/types/news.ts`).

## Структура

- `src/components/NewsBlock/` — компонент и его части (карточка, пагинация, skeleton, empty).
- `src/hooks/useNews.ts` — загрузка и кэширование данных.
- `src/types/news.ts` — типы данных.
